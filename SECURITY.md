# Security Policy

# Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x-{beta, alpha}.  | :x:              |

# Reporting a Vulnerability in the Client

Report any discovered vulnerability to the TeaClient Team `security@teaclient.com`. Your report will be acknowledged within 5 days, and a team member will be assisgned to fix this Vulnerability. The TeaClient Team is happy to report the progress vulnerability. We may seek other guidance from the community to help fix this vulnerability in a safe manner and private as possible. Once this vulnerability is fix it will be announced to the community publicly via a published cve once a fix has been applied.

# Reporting a Vulnerability in Open-Source
